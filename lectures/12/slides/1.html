Тема 12.1: Ринок хмарної безпеки

Розмір ринку

Основні сегменти

  • CSPM/CNAPP: ~20% (Wiz, Palo Alto, Orca)
  • Cloud IAM: ~18% (Okta, CyberArk)
  • CASB: ~15% (Netskope, Zscaler)
  • Cloud SIEM/XDR: ~15% (Splunk, Microsoft)
  • CWPP: ~12% (CrowdStrike, SentinelOne)

Cloud-Native Vendors

  • Wiz: $10B valuation, agentless CNAPP
  • Orca: SideScanning, unified platform
  • Lacework: Anomaly-based, Polygraph
  • Aqua: Container-first, shift-left
  • Snyk: Developer security leader

Enterprise Vendors

  • Palo Alto: Prisma Cloud (acquired RedLock)
  • CrowdStrike: Falcon Cloud Security
  • Microsoft: Defender for Cloud
  • Check Point: CloudGuard
  • Trend Micro: Cloud One

Тренди 2024-2025

  • Consolidation: CNAPP absorbing point tools
  • AI Integration: LLMs для triage, remediation
  • Shift-Left: Security в CI/CD
  • Identity-First: CIEM як критичний компонент
  • Platform Engineering: Security as product
