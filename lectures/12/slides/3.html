Тема 12.3: Програма безпеки та DevSecOps

Security Program Layers

  • Governance: Policies, standards, roles
  • Controls: Preventive, detective, corrective
  • Operations: Monitoring, response, maintenance
  • Framework: NIST CSF, CIS, CSA CCM
  • Continuous improvement cycle

Team Structure

  • Centralized: Dedicated team, consistency
  • Federated: Champions, scalability
  • Hybrid: Central + embedded (recommended)
  • Roles: Architect, Engineer, Analyst, DevSecOps
  • Security Champions Program
