Тема 7.2: Основні функції CSPM

5 ключових функцій CSPM

  • 1. Asset Discovery & Inventory
  • 2. Configuration Assessment
  • 3. Risk Prioritization
  • 4. Compliance Monitoring
  • 5. Remediation

1. Asset Discovery

  • Compute: VMs, containers, serverless
  • Storage: Object storage, databases
  • Network: VPCs, security groups, load balancers
  • Identity: IAM users, roles, service accounts
  • Real-time inventory через API polling

2. Configuration Assessment

  • CIS Benchmarks: Стандартні перевірки для AWS/Azure/GCP
  • Provider best practices: AWS Well-Architected
  • Custom policies: Організаційні стандарти
  • Compliance frameworks: PCI DSS, HIPAA, SOC 2

3. Risk Prioritization

  • Severity: Public exposure vs internal misconfiguration
  • Exploitability: Наскільки легко експлуатувати
  • Asset criticality: Production vs dev environment
  • Attack path: Чи є шлях до критичних даних

4. Compliance Monitoring

  • Real-time compliance dashboards
  • Automated reports для аудиторів
  • Gap analysis: що потрібно виправити
  • Evidence collection для audit trails

5. Remediation

  • Manual: Детальні інструкції для операторів
  • Semi-automated: CLI commands, IaC patches
  • Fully automated: Auto-fix за правилами
  • Preventive: Блокування в CI/CD

Розширені можливості

  • CIEM: IAM permissions analysis, unused privileges
  • DSPM: Sensitive data discovery, encryption status
  • Network: Topology visualization, exposure analysis
  • Attack Path: Моделювання можливих атак
