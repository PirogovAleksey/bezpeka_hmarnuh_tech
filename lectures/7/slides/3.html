Тема 7.3: Архітектура та інтеграція CSPM

Компоненти CSPM

  • Data Collection: API інтеграція з хмарами
  • Normalization: Єдина модель даних
  • Policy Engine: Правила перевірки
  • Risk Analysis: Scoring та prioritization
  • Presentation: Dashboards, alerts, reports

Data Collection Layer

  • API Integration: AWS, Azure, GCP APIs
  • Read-only access: Тільки читання конфігурацій
  • Event-driven: CloudTrail, Activity Log для real-time
  • Agentless: Не потребує агентів

Policy Engine

  • Бібліотека правил (CIS, NIST, custom)
  • OPA/Rego: Популярний policy language
  • Policy versioning та testing
  • Import/export для sharing

Deployment моделі

  • SaaS: Vendor хостить, швидкий старт, auto-updates
  • Self-hosted: Повний контроль, складніший maintenance
  • Hybrid: Control plane SaaS, data залишається on-prem

CI/CD Integration (Shift-Left)

  • Сканування IaC до deployment
  • Terraform: tfsec, checkov, terrascan
  • CloudFormation: cfn-lint, cfn-nag
  • PR checks: блокування з misconfigurations

SIEM/SOAR Integration

  • SIEM: Forwarding alerts до Splunk, Sentinel
  • SOAR: Triggering playbooks для auto-remediation
  • Correlation з іншими security events
  • Unified incident response

Інші інтеграції

  • Ticketing: Jira, ServiceNow auto-creation
  • Communication: Slack, Teams, PagerDuty
  • CMDB: Asset context enrichment
  • API: Custom integrations
