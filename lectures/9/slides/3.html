Тема 9.3: AI для автоматизації Security Operations

Intelligent Alert Triage

  • Prioritization: ML scoring на основі історії
  • Correlation: Групування related alerts
  • FP Reduction: Learning від analyst feedback
  • Результат: Фокус на реальних threats

SOAR з AI Enhancement

  • Auto-actions: Isolate, block, revoke
  • Dynamic playbooks: Вибір на основі incident type
  • Confidence-based: High confidence = auto-remediate
  • Learning: Нові playbooks з analyst actions

Security Copilots (LLMs)

  • Microsoft Security Copilot: GPT-4 для security
  • "Summarize this incident"
  • "What actions should I take?"
  • "Generate investigation report"
  • Natural language → KQL/SQL queries

LLM Use Cases

  • Alert summarization: Raw → human-readable
  • Investigation: Рекомендації next steps
  • Reporting: Automated incident reports
  • Code analysis: Пояснення malicious scripts

LLM Challenges

  • Hallucinations: Генерація неправди
  • Privacy: Що йде в cloud LLM?
  • Prompt injection: Маніпуляція через input
  • Over-reliance: Надмірна довіра AI

Vulnerability Prioritization

  • CVSS alone: Тисячі Critical/High
  • EPSS: Exploit Prediction Scoring
  • Threat intel: Actively exploited?
  • Context: Internet-facing? Production?
  • Результат: Фокус на реально exploitable
