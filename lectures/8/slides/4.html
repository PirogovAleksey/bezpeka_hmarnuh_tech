Тема 8.4: Ринок CNAPP та впровадження

Gartner Magic Quadrant Leaders

  • Wiz: Agentless, graph-based, $10B+ startup
  • Prisma Cloud: Most comprehensive, enterprise
  • Microsoft Defender: Azure-native, M365 integration
  • CrowdStrike: EDR heritage, threat intel

Інші ключові гравці

  • Orca: SideScanning, 100% agentless
  • Lacework: Behavior-based anomaly detection
  • Aqua: Container-focused, open source (Trivy)
  • Sysdig: Runtime security, Falco

Критерії вибору

  • Coverage: Які clouds, K8s, serverless?
  • Deployment: Agentless vs agent?
  • Depth: CSPM, CWPP, CIEM sophistication?
  • Attack paths: Якість risk prioritization?
  • Integrations: SIEM, SOAR, CI/CD, ticketing?
  • TCO: Pricing model, implementation cost?

Фази впровадження

  • Assessment: 2-4 тижні — inventory, requirements
  • POC: 3-4 тижні — vendor evaluation
  • Deployment: 4-6 тижнів — production rollout
  • Operations: Ongoing — optimization, metrics

Best Practices

  • Visibility first: Не enforcement одразу
  • Attack paths focus: 10 paths > 1000 findings
  • Enable DevOps: Self-service, clear guidance
  • Shift-left: CI/CD integration early
  • Measure: MTTD, MTTR, coverage

Майбутнє CNAPP

  • AI/ML: Intelligent prioritization
  • Runtime: eBPF, real-time behavior
  • AppSec: SAST/DAST, API security integration
  • Multi-cloud native: Cross-cloud attack paths
  • Gartner: 60% enterprises → CNAPP до 2026

Ключові висновки

  • CNAPP = Consolidation of cloud security tools
  • Attack path analysis — головна диференціація
  • Context + correlation > isolated findings
  • Dev + Sec + Ops — єдина платформа для всіх
  • Market is maturing — CNAPP стає стандартом
