CSPM

  • Configuration assessment для cloud infrastructure
  • Compliance monitoring (CIS, SOC 2, PCI DSS)
  • Asset inventory та visibility
  • Детально — Лекція 7

CWPP

  • Vulnerability Management: CVEs у workloads
  • Runtime Protection: Behavior monitoring
  • Malware Detection: Антивірус для cloud
  • File Integrity: Виявлення змін
  • Network: Microsegmentation

CIEM

  • Permission Analysis: Хто має доступ до чого
  • Least Privilege: Excessive permissions
  • Unused Access: Права без використання
  • Cross-account: Mapping доступу
  • Service Accounts: Non-human identities

Container & IaC Security

  • Image Scanning: CVEs, malware, secrets
  • Registry Security: Захист registries
  • K8s Security: RBAC, Network Policies
  • IaC Scanning: Terraform, CloudFormation
  • CI/CD Integration: Pre-deployment checks

DSPM

  • Data Discovery: Де зберігаються дані
  • Classification: PII, PHI, PCI, secrets
  • Data Flow: Куди переміщуються
  • Encryption: At-rest та in-transit
