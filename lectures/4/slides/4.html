Runtime Security: Навіщо?

  • Статичне сканування не знаходить все
  • Zero-day вразливості
  • Атаки відбуваються в runtime
  • Останній рубіж захисту

System Call Monitoring

  • Моніторинг syscalls контейнера
  • Виявлення: reverse shell, file access
  • Falco: CNCF project, rules-based
  • Tetragon: eBPF-based від Cilium

eBPF для безпеки

  • Програми в ядрі Linux без модифікації
  • Переваги: низький overhead, глибока видимість
  • Можливість enforcement (блокування)
  • Інструменти: Tetragon, Tracee, Falco

Image Signing

  • Запускати тільки довірені образи
  • Cosign (Sigstore): підписування образів
  • Admission controller: перевірка підпису
  • GCP Binary Authorization

CWPP рішення

  • Commercial: Prisma Cloud, CrowdStrike, Aqua, Wiz
  • Open Source: Falco, Trivy, Kubescape
  • Cloud-native: AWS Inspector, Defender for Cloud
