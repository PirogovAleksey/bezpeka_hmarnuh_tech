Тема 10.3: Управління ключами (Key Management)

Чому управління ключами (Key Management) критичне?

  • Компрометація ключа = компрометація всіх даних
  • Втрата ключа = втрата доступу до даних
  • Ротація ключів (Key rotation) для long-term security
  • Encryption без управління ключами (key management) — ілюзія безпеки

Cloud KMS Services

  • AWS KMS: Integration з 100+ сервісів, CloudTrail
  • Azure Key Vault: Keys, secrets, certificates
  • GCP Cloud KMS: Hierarchical, HSM support
  • Automatic key rotation available

Конвертне шифрування (Envelope Encryption)

Ротація ключів (Key Rotation)

  • Automatic: AWS KMS annual rotation
  • Manual: Create new, re-encrypt, deprecate old
  • Rotate annually minimum
  • Immediate при suspected compromise
  • Automate where possible

BYOK vs HYOK

Найкращі практики (Best Practices)

  • Use managed KMS services
  • Enable automatic rotation
  • Separate keys per environment/purpose
  • Audit all key operations
  • Test key recovery procedures
