Тема 11.1: Cloud Threat Landscape

Чому хмари — привабливі цілі?

  • Concentration of valuable data
  • Complex attack surface
  • Skills gap — брак кваліфікованих спеціалістів
  • Shared responsibility confusion
  • Один exploit = тисячі жертв

CSA Top Threats 2024

  • #1: Misconfiguration & Inadequate Change Control
  • #2: Identity & Access Management
  • #3: Insecure Interfaces & APIs
  • #4: Lack of Cloud Security Strategy
  • #5: Insecure Third-Party Resources

Cloud-Specific Attack Techniques

  • SSRF: Server-Side Request Forgery → metadata
  • IMDS Exploitation: 169.254.169.254
  • Cross-Account Attacks: Trust policy abuse
  • Container Escape: Breakout to host
  • Serverless Injection: Event data poisoning

Статистика інцидентів

  • 45% — web application attacks
  • 22% — credential theft
  • 17% — misconfigurations
  • Mean time to detect: 197 days
  • Average cost: $4.45M (IBM 2023)

Threat Actors

  • APT groups: Nation-state (APT29, APT41)
  • Ransomware gangs: LockBit, BlackCat
  • Cryptominers: Resource hijacking
  • Hacktivists: Politically motivated
  • Insiders: Employees, contractors
