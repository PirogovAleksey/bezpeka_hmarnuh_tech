Log4Shell (2021) — Overview

  • CVE-2021-44228 — CVSS 10.0
  • Apache Log4j — найпопулярніша Java logging
  • JNDI lookup → Remote Code Execution
  • Мільярди vulnerable систем
  • "Most critical vulnerability of the decade"

Log4Shell — Attack Vector

Log4Shell — Impact

  • AWS, Azure, GCP — всі affected
  • Minecraft servers — перші атаки
  • iCloud, Steam, Twitter vulnerable
  • Cryptominers deployed within hours
  • Ransomware campaigns followed

Twitch Breach (2021)

125GB source code leaked

Streamer payouts, internal tools, credentials exposed

  • Причина: Server misconfiguration
  • Lesson: Data classification + access control

Uber Breach (2022)

  • Social engineering → MFA fatigue
  • Hardcoded credentials in scripts
  • Full access to internal systems
  • Slack, AWS, GCP compromised
  • Lessons: No hardcoded secrets, MFA review

LastPass (2022-2023)

  • Developer machine compromised
  • Cloud backup storage accessed
  • Encrypted vaults stolen
  • Weak master passwords at risk
  • Lessons: Endpoint security, encryption strength
